## About

I'm a full-stack developer who builds web applications, with a focus on AI integration. Most of my work runs on **Next.js, React, and TypeScript**, backed by **Node.js, Express, and Prisma**.

Recent projects include a tool for asking questions about any codebase, a real-time AI video-meeting platform, and a multi-tool AI content app. I've also published open-source packages for the React ecosystem.
